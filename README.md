# IOTA Workshop

IOTAのJSライブラリーを使ってみよう.

### Usage

```bash
git clone https://github.com/solareenlo/iota-workshop.git
cd iota-workshop
npm i
node src/1-hello-world.js
```
以上を行うことで, 接続しているIOTAノードの情報が標準出力に表示されると思います.  
同様に色々と試してみてください.
